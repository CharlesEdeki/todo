# Todo App

A simple todo application built with React and Vite. It uses JSONPlaceholder as a mock backend to demonstrate full CRUD-like interactions — listing, viewing, creating, and deleting todos.

---

## What it does

- Fetches and displays 50 todos from the JSONPlaceholder API in a paginated table
- Search todos by title and filter by status (all, done, pending)
- Add new todos through a modal form
- View individual todo details on a separate page
- Delete todos with a confirmation step before anything is removed
- Shows live stats for total, done, and pending counts

---

## Tech stack

- React 18
- Vite
- Axios
- JSONPlaceholder (mock REST API)
- Plain CSS with custom properties for theming
